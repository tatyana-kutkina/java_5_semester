# java_5_semester
task 1 - Реализовать программу на Java, которая вычисляет все возможные варианты размена данной суммы на купюры. Также следует вычислить общее количество уникальных комбинаций.  

task 2 - 
1. Реализовать на Java линейный список на основе массива. В качестве элемента принимать любой объект. 
2. На основе списка реализовать стек. 
3. При выполнении 1 имплементировать интерфейс java.util.List. При реализации стека расширить класс java.util.Stack. Методы, не обозначенные в задаче, реализовать по желанию.

task 3 -
1. Реализовать на Java хеш-таблицу. Использовать методы .equals и .hashCode класса Object для работы с таблицей.
2. На основе хеш-таблицы реализовать
3. При выполнении 2 реализовать интерфейс java.util.Map

task 4 -
Реализовать на Java программу, которая записывает список файлов и папок в заданной директории в файл. Следует совершить рекурсивный обход всего поддерева в файловой системе. Путь к директории и путь к целевому файлу передать через аргументы программы. 
* Предусмотреть возможность использования относительных путей

task 5 - 
Реализовать на Java пул ресурсов. В качестве ресурса должен предоставляться один из объектов (лучше передавать через Generic):
Поток
Файл (для записи лога)
Объект произвольного класса

Предусмотреть возможность задания фиксированного размера пула, а также возможности его увеличения. 
Создавать ресурсы по необходимости, выключать после n секунд бездействия. 

В дополнение к пулу ресурсов реализовать программу, использующую его для решения 10000 квадратных уравнений и записи их в файлы.
